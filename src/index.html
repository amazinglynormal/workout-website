Introduction

This is a general size and strength development program for novices. Its goal is to increase strength on the major compound lifts while also developing overall muscle size. Alowing you to have a decent base from which to explore more specific training and goals like powerlifting or bodybuilding for example.

While I am not a personal triainer or exercise scientist, I am putting this program online to allow anyone looking for a free program to copy it. I got great results from this program and hope it also benefits you.

Program

UPPER A
Bench Press 3 x 3-5
Pull Ups 3 x 3-5 Depending on experience level this could be assisted, bodyweight or weighted.
Pause 30° Incline DB Bench 3 x 8-12 Pause for 2 seconds at the bottom.
Upper Back Helms Row 3 x 6-10
Seated DB OHP 3 x 6-10
EZ Bar Curl 3 x 10-15
Tricep Pushdown 3 x 10-15
DB Lateral Raises 3 x 10-15
LOWER A
Squats 3 x 3-5
RDL 3 x 6-10
Leg Press 3 x 6-10
Hip Thrust 3 x 8-12
Calf Raises 3 x 8-12
DB Rear Delt Flye 3 x 10-15
Abs 3 x 10-15 Whatever variation you prefer or is available that you can progressively overload over time.
UPPER B
OHP 3 x 3-5
Barbell Bent-Over Row 3 x 6-10
Dips (Chest Focused) 3 x 8-12 Depending on experience level this could be assisted, bodyweight or weighted.
Neutral-Grip Lat Pulldown 3 x 6-10
Incline DB Flye 3 x 10-15 Whatever incline you prefer. I use a 45° angle.
Hammer Curl 3 x 10-15
EZ Bar Skullcrusher 3 x 10-15
DB Lateral Raises 3 x 10-15
LOWER B
Deadlift 3 x 3-5
Front Squats 3 x 6-10
Leg Extension 3 x 10-15
Leg Curls 3 x 6-10 Whatever variation you prefer or is available to you.
Calf Raises 3 x 8-12
DB Rear Delt Flye 3 x 10-15
Abs 3 x 10-15 Whatever variation you prefer or is available that you can progressively overload over time.
